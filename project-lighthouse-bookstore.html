Lighthouse Bookstore

C#/WinForm Bookstore Management System

Accomplishments

This bookstore management system handles inventory, sales orders and transactions for a small bookstore. As the lead developer, my responsibilities included designing the database schema and user interface, as well as implementing core algorithms.

  • Built a fully functional system that manages inventory of hundreds of books, processes orders and tracks transactions for hundreds of customers amd stores records for several employees
  • Designed user-friendly screens for key functions like sales, inventory checks, and reports
  • Created a robust SQL Server database that handles exceptions and invalid inputs gracefully and features low coupling and high cohesion through role-based access privileges

By using database technology, the system was able to streamline bookstore operations such as inventory tracking, sales analytics, and order management.
